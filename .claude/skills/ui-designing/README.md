# UI Designing Skill for Scientific Calculator

This skill provides comprehensive guidance for creating super modern, aesthetic, and highly usable web UI for scientific calculator projects.

## Features

- Modern, clean design principles with proper spacing and visual hierarchy
- Comprehensive styling guidance using Tailwind CSS
- Complete JavaScript implementation for calculator functionality
- Accessibility best practices and keyboard navigation support
- Responsive design patterns for different screen sizes
- Animation and interaction feedback systems

## Usage

This skill is automatically triggered when Claude needs to:
- Design UI layouts for scientific calculators
- Create styling and visual aesthetics
- Structure components with modern design principles
- Implement responsive design patterns
- Build accessible and usable calculator interfaces

## Contents

- `SKILL.md` - Main skill instructions and guidelines
- `references/patterns.md` - UI design patterns and examples
- `references/javascript-implementation.md` - JavaScript implementation guide
- `scripts/generate-calculator.js` - Script to generate a complete calculator UI

## Key Design Principles

1. **Clean Layout** - Generous spacing and proper white space usage
2. **Visual Hierarchy** - Prominent display area with grouped buttons
3. **Modern Aesthetics** - Minimalist design with subtle animations
4. **Accessibility** - Proper ARIA labels and keyboard navigation
5. **Responsive Design** - Works on all device sizes

The skill emphasizes creating beautiful, functional, and accessible scientific calculator UIs with modern aesthetics and excellent usability.
